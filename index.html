Meet the Zeditors

We are all employed by IBM Australia as System Z technical specialists, and share a passion for building things on our z/OS test rigs. (we all use IBM Z Development and Test software to run z/OS systems as a virtualized guest of Linux on some beefy x64 systems at home).

As our recipe notebooks for getting things done grew larger and larger, we wanted a permenant place to safely store them and share them. Github was the obvious repository for our code and documentation, and Github Pages turns out to be a great way to publish it. This website will grow over time, so please drop by from time to time to find new recipes.
